#  Sistema de Historia Clínica (Loom)

## Características

- Gestión de pacientes, médicos y EPS
- Centralización de la historias clínicas de los pacientes 
- Registro de consultas médicas
- Exámenes clínicos
- Prescripción de medicamentos
- Manejo de Traslado de pacientes entre EPSs 

## Estructura del proyecto

scripts/
- 01_Tablas.sql
- 02_Inserciones.sql
- 03_Consultas.sql
- 04_Procedimientos_Triggers.sql

Documentos/
- Propuesta Dominio y justificación

## Consultas

- Historial clínico por paciente
- EPS con mayor número de pacientes
- Enfermedad más frecuente por EPS

## Tecnologías usadas por el momento

- PostgreSQL
- SQL
